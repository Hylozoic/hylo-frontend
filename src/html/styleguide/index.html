Mixin Usage
.user-avatar(@size) add properties to display a user avatar of the supplied size
.user-avatar-block() add properties to display the user avatar as a block
.add-avatar-margin(@size, @use-properties) add a margin of the supplied size to fit an avatar to the elements left
.remove-link-display(@color) remove the settings that make an element look like a link. Resets to the default text color unless another is supplied.

Classes

Class Name Usage Options
.user-avatar-* display a user avatar as a background image xxs, xs, sm, lg
.avatar-margin-* add a margin to fit an absolute positioned avatar to the left of any html element xs, sm. med, lg
